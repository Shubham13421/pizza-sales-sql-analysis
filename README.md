# Pizza Sales Data Analysis (SQL Project)

## Project Overview
This project analyzes pizza sales data to uncover business insights such as revenue trends, customer ordering patterns, and product popularity using SQL.

## Tools Used
- SQL (MySQL)
- Canva

## Dataset
The dataset consists of four tables:
- orders
- order_details
- pizzas
- pizza_types

## Business Questions Solved
- Retrieve the total number of orders placed
- Calculate total revenue generated
- Identify the highest-priced pizza
- Identify the most common pizza size ordered
- Determine peak order hours
- Calculate average number of pizzas ordered per day
- Analyze revenue contribution by pizza category

## Key Insights
- Total orders: **21,350**
- Most ordered pizza size: **Large (L)**
- Peak order time: **11 AM – 1 PM and 5 PM – 7 PM**
- Average daily orders: **138**
